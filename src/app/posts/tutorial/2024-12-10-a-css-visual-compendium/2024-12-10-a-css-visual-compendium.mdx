What is this, you ask? It's a way for me to review CSS silly. 

No, but actually, it's a tool to see how different CSS properties work in real-time
using an <span className="bg-[#D3D3D3] border-[1px] border-black">interactive element</span> that can be styled with sliders, inputs, etc.

So you can really see how changing the CSS properties' values affects the element.

A core concept in CSS is whether styles are applied using *modules* (internal or external) or *inline*
- modular styles are imported from an external .css file
- inline styles are defined in the body of an element: e.g. `<div styles="padding: 2px">`
